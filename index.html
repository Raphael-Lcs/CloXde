Quick Start

快速开始

安装命令、下载入口与文档都集中在这里，适合快速把 CloXde 接入你的协作流程。

terminal
git clone https://github.com/Raphael-Lcs/CloXde
cd CloXde
pnpm install
pnpm dev

1. 从 Releases 下载客户端，或从源码运行（git clone + pnpm install && pnpm dev）。

2. 在项目目录中启动协作环境，让 AI 团队读取上下文并执行任务。

3. 按需提交 Issue、PR 或查阅 README 获取更多使用细节。
